# 🚲 Snap Bikes Trip Data Analysis

## Project Overview
This project analyzes bike trip usage patterns using Tableau.

## Dashboard Highlights
- Total trips analyzed: 50,000
- Station distribution analysis
- Start vs End station trends
- Trip duration by gender
- Time-based trip patterns

## Tools Used
- Tableau
- Data Cleaning
- Data Visualization

## Key Insights
- Peak trip hours occur in evening commute times.
- Certain stations dominate start/end activity.
- Trip duration varies by gender category.
